# Installation

## Stable release

To install Secret Santa, run this command in your terminal:

```sh
uv add python-boilerplate
```

Or if you prefer to use `pip`:

```sh
pip install python-boilerplate
```

## From source

The source files for Secret Santa can be downloaded from the [Github repo](https://github.com/diarmuidhenry/secret_santa).

You can either clone the public repository:

```sh
git clone git://github.com/diarmuidhenry/secret_santa
```

Or download the [tarball](https://github.com/diarmuidhenry/secret_santa/tarball/master):

```sh
curl -OJL https://github.com/diarmuidhenry/secret_santa/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd secret_santa
uv pip install .
```
